# meiqdad633.github.io # Meiqdad Hassanni's Portfolio This repository contains HTML and CSS code for Meiqdad Hassanni's personal portfolio webpage. The webpage provides information about Meiqdad's background, skills, experiences, and contact details. ## Table of Contents - [HTML Structure](#html-structure) - [CSS Styles](#css-styles) - [Usage](#usage) - [Contributing](#contributing) - [License](#license) ## HTML Structure ### Header - `h1` displays the name. - `p` with the ID `position` indicates the position. ### Summary Section - Contains a link for a "Dark Mode" toggle. - `h2` marks the start of the summary. - `p` with the class `sumtext` displays the introductory text and an image. ### Experience Section - `h2` marks the start of the experience section. - An unordered list (`ul`) with the ID `experience` lists experiences. Each list item (`li`) contains experience details. ### Education Section - `h2` marks the start of the education section. - An unordered list (`ul`) with the ID `education` lists educational qualifications. Each list item (`li`) contains education details. ### Skills Section - `h2` marks the start of the skills section. - A table with the ID `skills` displays a tabular representation of skills. ### Footer - Contains contact details, links to social profiles, and a "Contact Form" section. - The "Contact Form" includes fields for name, email, and a message, along with a "SEND" button. ## CSS Styles - Defines the overall styling for the webpage. - Specifies font families, sizes, margins, borders, background colors, and text alignments for various HTML elements. - Defines specific styling for the header, list items, links, tables, input fields, and more. ## Usage 1. Clone this repository to your local machine. 2. Open the `index.html` file in a web browser to view the portfolio webpage. 3. Explore different sections to learn more about Meiqdad Hassanni. 4. Use the contact form to send messages or inquiries. ## Contributing Contributions to this repository are generally not accepted since it represents a personal portfolio.
